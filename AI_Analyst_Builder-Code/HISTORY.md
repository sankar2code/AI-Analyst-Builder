## About This Project

AI Analyst Builder is a personal toolkit for running end-to-end product analytics using Claude Code. It handles framing, data exploration, root cause analysis, storytelling, and slide deck creation — all driven by natural language queries over your own data.

The agents, skills, and pipeline are all markdown files you can read and modify. Nothing is hidden.
